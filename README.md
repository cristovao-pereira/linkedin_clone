
# LinkedIn-Style Profile (HTML + CSS)

This repository contains a **static LinkedIn-style profile page** built using **only HTML and CSS**.

The project is designed as a **learning exercise** for frontend students to practice real-world layout structure, semantic HTML, and clean CSS organization, without JavaScript or backend dependencies.
